# 🏫 智联校园 AI 助手（RAG 智能问答系统）

基于 FastAPI + Streamlit + ChromaDB + 硅基流动大模型 构建的校园知识库问答系统。

## 🌟 项目亮点
- 前后端分离架构
- 支持 PDF 上传与解析
- RAG 检索增强生成
- 知识库向量存储
- 基于 DeepSeek-V3.2 大模型
- 完整可部署、可用于简历项目

## 🛠 技术栈
- 后端：FastAPI
- 前端：Streamlit
- 向量库：ChromaDB
- 大模型：SiliconFlow DeepSeek-V3.2
- 文档解析：PyPDF
- RAG 全流程：文本分块 → 向量化 → 检索 → 生成回答
